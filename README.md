# Resuflow

Simple resume builder. Free, build as many resumes as you want - all data stays local in your browser.

## Features

- 4 resume templates (Classic Clear, Slate Sidebar, Compact ATS, Obsidian Edge)
- Customize fonts, colors, spacing, and page format
- Download as PDF
- Undo / redo
- All data stored locally - no account, no server, no tracking

## Usage

```bash
npm install
npm run dev
```

## Release

```bash
npm run release 1.0.0
```

Bumps version, commits, tags, and pushes. GitHub Actions deploys to GitHub Pages on tag push.
